Active Directory Security Audit

Complete security audit of an enterprise Active Directory infrastructure with vulnerability identification and hardening proposals.

Scroll

This project involved conducting a comprehensive security audit of an Active Directory infrastructure for a medium-sized company. The goal was to identify security flaws and propose corrective measures.

The audit covered GPO configuration, access rights, privileged account management, and compliance with Microsoft security best practices.

Results and recommendations

  • Privileged Account Management Implementation of PAW (Privileged Access Workstation) and rights separation
  • Monitoring and Logging Advanced audit configuration and centralization of security logs
