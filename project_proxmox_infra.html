Infrastructure Proxmox & AD

Infrastructure complète virtualisée sur Proxmox avec déploiement d'un serveur Active Directory, configuration d'un switch manageable et segmentation réseau via VLANs pour créer un environnement de test isolé et sécurisé.

Scroll

Contexte du projet

Mise en place d'une infrastructure complète de test et développement basée sur Proxmox VE. Le projet inclut le déploiement d'un serveur Active Directory pour gérer l'authentification centralisée et les stratégies de groupe.

L'infrastructure réseau comprend un switch manageable configuré avec plusieurs VLANs pour segmenter les environnements (production, test, développement). Cette architecture permet de tester des configurations en isolation complète avant déploiement en production et facilite les exercices de simulation d'incidents.

Fonctionnalités principales

  • Plateforme de virtualisation complète Serveur Proxmox avec gestion des VMs/containers, snapshots, migrations à chaud et clustering haute disponibilité
  • Active Directory intégré Serveur AD avec DNS, DHCP, GPO et authentification centralisée pour tous les postes et services de l'infrastructure
  • Segmentation réseau avancée Configuration switch avec VLANs multiples pour isoler production/test/développement avec routage inter-VLAN contrôlé
