Automatisation avec Ansible

Déploiement automatisé d'infrastructure serveur avec Ansible pour standardiser les configurations et accélérer les déploiements.

Scroll

Mise en place d'une solution d'automatisation avec Ansible pour le déploiement et la configuration de serveurs Linux. L'objectif était de réduire le temps de provisioning et d'assurer la cohérence des configurations.

Le projet inclut la création de playbooks pour l'installation de services, la configuration réseau, et l'application de politiques de sécurité.

Technologies utilisées

Ansible

Core & Tower

Linux

Ubuntu & CentOS

Python

Modules custom

Résultats et bénéfices

  • Réduction du temps de déploiement Passage de 2h à 15 minutes pour un déploiement complet
  • Standardisation des configurations Infrastructure as Code garantissant la cohérence
  • Gestion centralisée Contrôle version et rollback facilités avec Git
