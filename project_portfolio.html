Portfolio Interactif

Site portfolio personnel développé entièrement en HTML/CSS/JavaScript vanilla avec animations fluides, dashboard d'expérience professionnelle de type "réseau neuronal", système de navigation dynamique et design responsive moderne.

Scroll

Contexte du projet

Développement d'un site portfolio personnel moderne et interactif pour présenter mes compétences, projets et expériences professionnelles. Le site intègre des animations CSS avancées, un dashboard d'expérience avec visualisation de type "réseau neuronal", et une navigation fluide.

Le projet a été réalisé entièrement en technologies web natives (HTML5, CSS3, JavaScript vanilla) sans framework, avec un focus sur les performances, l'accessibilité et le responsive design.

Technologies utilisées

Design

Fonctionnalités principales

  • Dashboard expérience "réseau neuronal" Page d'expérience interactive avec animation de particules, nœuds connectés et design futuriste
  • Animations fluides et responsive Transitions CSS avancées, scroll indicators, animations d'apparition progressives et design mobile-first
  • Système de navigation dynamique Menu contextuel, switcher de langue FR/EN, accordions interactifs et formulaire de contact fonctionnel
