Active Directory Automation

Suite of PowerShell scripts to automate Active Directory administration tasks: automatic domain joining, GPO configuration, user creation and security group management.

Scroll

Project Context

Development of a PowerShell script collection to automate repetitive Active Directory administration tasks. The goal was to save time and reduce human errors during common operations.

The scripts cover four main areas: automatic domain joining of workstations with network configuration, GPO deployment and modification, bulk user creation from CSV files, and automated security group management with permission assignment.

Key Features

  • Automatic domain joining Scripts to add workstations to the domain with DNS configuration, automatic reboot and connectivity verification
  • User and group management Bulk user creation from CSV, password generation, automatic security group assignment
  • GPO automation Scripts for creating and modifying Group Policies with OU-based application, GPO backup and restore
